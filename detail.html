Valorant apresenta erro 62 e está fora do ar nesta quarta-feira (1)

Valorant está off? Usuários dizem que jogo caiu e Server Status indica problemas

Roberto da silva Jr.

01/09/2021 às 09h56

Valorant está fora do ar nesta quarta-feira (1) por conta de problemas de conexão com o servidor. De acordo com o serviço oficial de Status da Riot Games, a falha impede de realizar login e é de nível crítico. A desenvolvedora afirma que está trabalhando na correção do erro.

Os usuários indicam que o erro 62 e a mensagem "houve um erro ao se conectar com a plataforma Valorant" aparecem ao tentar logar no game.

O Jovem Nerd entrou em contato com a Riot Games para saber se há previsão de retorno e atualizará esta notícia quando tiver resposta.

De acordo com o site Downdetector, que monitora plataformas online, os problemas começaram por volta das 6h30 e, três horas depois, Valorant recebeu 62 reclamações. As notificações envolvem problemas de conexão com os servidores e, consequentemente, falha para realizar o login no game.

O erro 62 está ligado a falhas de conexão com a Internet. Normalmente, é preciso reiniciar o cliente do game para solucionar a questão. No entanto, a ação não deve funcionar para esta situação. No Server Status de Valorant, a primeira notificação indicava problemas no saguão por volta de 9h30. No entanto, às 9h56, o serviço foi atualizado para "problemas de login". Dessa forma, os usuários não podem jogar e devem aguardar a normalização do serviço.
